# Product card images

Drop an image in this folder using the exact filename listed below for
each game. The card picks it up automatically — no code changes needed.
If a file is missing, that card just keeps showing its gradient
placeholder, so you can add these one at a time.

Recommended size: 560×360px or larger, landscape, JPG or PNG.
